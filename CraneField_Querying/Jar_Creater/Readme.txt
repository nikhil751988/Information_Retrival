Steps to compile and run the program

Below are the commands and instructions to run the program

1. Copy the java files and the shell script provided with this zip to the same location and give the shell script execute    permission

	chmod 744 jarCreater.sh


2. Run the shell script to launch the program:

	./jarCreater.sh

	-This shell script compiles all the files
	-Creates the executable jar file
	-Run the executable jar file

3. The jar will ask for the following details:

	Enter source location of the Cranfield:
	Ex./people/cs/s/sanda/cs6322/Cranfield 

	Enter the query file with location:
	Ex./people/cs/s/sanda/cs6322/hw3.queries

The program will run and the output is printed into Statistics.txt file in the present directory.
